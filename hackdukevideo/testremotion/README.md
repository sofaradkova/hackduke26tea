# Remotion Test Project

Tiny test project using [Remotion](https://remotion.dev/) + React + TypeScript.

## Quick start

```bash
npm install
npm run dev
```

Then open Remotion Studio in your browser and preview `TestComposition`.

## Render a video

```bash
npm run render
```

Output file will be:

```txt
out/test-video.mp4
```

## Project structure

- `src/index.ts` – Registers the Remotion root
- `src/Root.tsx` – Declares compositions
- `src/TestComposition.tsx` – Demo animation
- `remotion.config.ts` – Remotion config
